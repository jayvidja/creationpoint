# 🧱 Tiles Design Gallery

A full-stack web application for showcasing modern tile designs with a clean, responsive, and interactive user interface.

---

## 🚀 Features

* 🖼️ Browse beautiful tile collections
* 📂 Category-based design filtering
* ⚡ Fast and responsive frontend
* 🔐 Backend API integration
* 📱 Fully mobile-friendly design

---

## 🛠️ Tech Stack

### Frontend

* React.js
* CSS / Tailwind

### Backend

* Node.js
* Express.js
* MongoDB

---

## 📁 Project Structure

Tiles_Design_Gallery/
│
├── frontend/     # React frontend
├── backend/      # Node + Express backend
│
└── README.md

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/your-username/Tiles_Design_Gallery.git
cd Tiles_Design_Gallery

---

### 2️⃣ Setup Backend

cd backend
npm install
npm run dev

---

### 3️⃣ Setup Frontend

cd frontend
npm install
npm run dev

---


## ▶️ Running the App

Frontend: http://localhost:5173
Backend: http://localhost:5000
